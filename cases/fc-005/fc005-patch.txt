Subject: [PATCH] FC-005: domain scope guard refusing out-of-corpus queries

Adds a pre-retrieval gate in submit_query that refuses queries
containing phrase anchors outside the Alberta OHS corpus scope
(emissions regulations, workers comp, tax, IT procurement).

Closes the FC-005 failure mode from KDAT-001B where a greenhouse
gas TIER query returned Part 36 mine gas chunks via semantic overlap.

Phrase-based to avoid the KDAT-086 over-refusal failure mode.
Mirrors the existing jurisdiction guard pattern. New refusal reason
code DOMAIN_OUT_OF_SCOPE. Audit chain HMAC preserved.

Validated locally:
- FC-005 query refused with DOMAIN_OUT_OF_SCOPE
- WCB claim query refused with DOMAIN_OUT_OF_SCOPE
- lockout/tagout query still approves
- confined space entry query still approves
- audit chain integrity preserved, sources empty on refusal

Demo-grade fix for TTW 2026-05-25. Full taxonomy-based two-stage
gate per ADR-002 remains scoped to KDAT-002.
