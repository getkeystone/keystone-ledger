# Contributing

This repository is the public milestone ledger for Keystone Applied Intelligence. It is a
read-only evidence reference — not an open-source project accepting code
contributions.

## What belongs here

- Milestone summaries derived from delivery commits in private implementation
  repositories
- Public-safe claims that are verifiable against stated evidence
- Corrections to factual errors in milestone pages (raise an issue)

## What does not belong here

- Raw internal commit logs
- Private hostnames, infrastructure details, or personal information
- Speculation or forward-looking claims presented as delivered capability
- Artifact files that have not passed publication safety review

## Raising an issue

Use GitHub Issues to flag:
- A factual inaccuracy in a milestone page
- A milestone status that appears incorrect
- A claim that exceeds stated evidence

Label your issue with one of: `factual-error`, `status-correction`, `overclaim`.
